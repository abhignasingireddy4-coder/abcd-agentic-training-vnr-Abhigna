## 📌 Assignment
Implement Cosine Search and other Search Algorithms in Python.

## 🔍 Algorithms Implemented
| Algorithm | Type | Time Complexity |
|---|---|---|
| Cosine Similarity Search | Vector-based | O(n) |
| TF-IDF Ranked Search | Statistical | O(n) |
| Jaccard Similarity Search | Set-based | O(n) |
| Linear Search | Sequential | O(n) |
| Binary Search | Sorted list | O(log n) |

## ▶️ How to Run
pip install -r requirements.txt
python search_algorithms.py
